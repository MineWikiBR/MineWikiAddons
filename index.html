Tutorial Completo: Instalar Addons no PC (Windows)

Método 1: Via Minecraft Marketplace

  1. Abra o Minecraft Bedrock Edition no seu PC
  2. Na tela principal, clique na aba "Mercado" (Marketplace) no topo
  3. Navegue pelos addons disponíveis usando as categorias ou barra de pesquisa
  4. Selecione o addon desejado e clique em "Comprar" ou "Baixar" (se for gratuito)
  5. O addon será baixado e instalado automaticamente
  6. Vá em "Jogar" → Selecione um mundo → "Editar" → "Pacotes de Recursos" ou "Pacotes de Comportamento"
  7. Ative o addon deslizando o botão para a direita
Este método é o mais seguro e recomendado para iniciantes

Método 2: Instalação Manual via .mcpack

  1. Baixe o arquivo .mcpack do addon de um site confiável
  2. Localize o arquivo baixado (geralmente na pasta "Downloads")
  3. Clique duas vezes no arquivo .mcpack
  4. O Windows perguntará com qual programa abrir → Selecione "Minecraft"
  5. O Minecraft abrirá automaticamente e importará o addon
  6. Uma mensagem de confirmação aparecerá quando a importação for concluída
  7. Vá em "Jogar" → Selecione um mundo → "Editar" → Ative o addon
Certifique-se de baixar addons apenas de fontes confiáveis para evitar vírus

Método 3: Instalação Manual nas Pastas do Sistema

Este método requer conhecimento de navegação em pastas do Windows

  1. Pressione Windows + R para abrir o Executar
  2. Digite o caminho abaixo e pressione Enter:
%localappdata%\Packages\Microsoft.MinecraftUWP_8wekyb3d8bbwe\LocalState\games\com.mojang
  1. Dentro dessa pasta, você encontrará:
resource_packs - Para pacotes de recursos (texturas, modelos, sons)
behavior_packs - Para pacotes de comportamento (mecânicas, entidades, itens)
  1. Se você baixou um arquivo .mcpack, renomeie-o para .zip e extraia
  2. Copie a pasta extraída para resource_packs ou behavior_packs (conforme o tipo)
  3. Feche completamente o Minecraft (certifique-se de que não está rodando em segundo plano)
  4. Abra o Minecraft novamente
  5. Vá em "Jogar" → Selecione um mundo → "Editar" → Ative o addon
Caminho alternativo se o acima não funcionar: C:\Users\[SEU_USUÁRIO]\AppData\Local\Packages\Microsoft.MinecraftUWP_8wekyb3d8bbwe\LocalState\games\com.mojang

Método 4: Instalação de .mcaddon

  1. Baixe o arquivo .mcaddon (contém tanto resource quanto behavior pack)
  2. Renomeie a extensão de .mcaddon para .zip
  3. Extraia o arquivo ZIP usando WinRAR, 7-Zip ou o próprio Windows
  4. Dentro da pasta extraída, você encontrará duas pastas:
behavior_packs - Contém o pacote de comportamento
resource_packs - Contém o pacote de recursos
  1. Abra o caminho do Minecraft (Método 3, passo 2)
  2. Copie o conteúdo de behavior_packs para a pasta behavior_packs do Minecraft
  3. Copie o conteúdo de resource_packs para a pasta resource_packs do Minecraft
  4. Feche e reabra o Minecraft
  5. Ative ambos os pacotes no mundo nas configurações

Método 5: Instalação em Mundo Específico

  1. Abra o Minecraft e vá em "Jogar"
  2. Selecione o mundo onde deseja instalar o addon
  3. Clique no ícone de "Editar" (lápis) ao lado do mundo
  4. Vá em "Abrir Pasta do Mundo"
  5. Uma pasta será aberta no Windows Explorer. Dentro dela, você verá:
behavior_packs - Para addons de comportamento do mundo
resource_packs - Para addons de recursos do mundo
  1. Copie a pasta do addon para a pasta correspondente
  2. Volte ao Minecraft e ative o addon nas configurações do mundo
Este método instala o addon apenas naquele mundo específico, não globalmente

Tutorial Completo: Instalar Addons no Mobile

Método 1: Via Marketplace

  1. Abra o Minecraft Pocket Edition no seu dispositivo
  2. Na tela principal, toque na aba "Mercado" (Marketplace) no topo
  3. Navegue pelos addons disponíveis usando as categorias ou barra de pesquisa
  4. Toque no addon desejado para ver detalhes
  5. Toque em "Baixar" (se gratuito) ou "Comprar" (se pago)
  6. O addon será baixado e instalado automaticamente
  7. Vá em "Jogar" → Selecione um mundo → Toque no ícone de "Editar"
  8. Vá em "Pacotes de Recursos" ou "Pacotes de Comportamento"
  9. Ative o addon deslizando o botão para a direita
Este método funciona perfeitamente em Android e iOS

Método 2: Instalação Manual via .mcpack

  1. Baixe o arquivo .mcpack no seu dispositivo Android usando o navegador
  2. Abra o aplicativo "Arquivos" ou "Meus Arquivos" (pode variar conforme o fabricante)
  3. Navegue até a pasta "Download" ou "Downloads"
  4. Localize o arquivo .mcpack que você baixou
  5. Toque e segure no arquivo .mcpack
  6. Selecione "Abrir com" ou "Compartilhar"
  7. Escolha "Minecraft" na lista de aplicativos
  8. O Minecraft abrirá automaticamente e importará o addon
  9. Uma mensagem de confirmação aparecerá quando a importação for concluída
  10. Vá em "Jogar" → Selecione um mundo → "Editar" → Ative o addon
Se o Minecraft não aparecer na lista, certifique-se de que o app está instalado e atualizado

Método 3: Instalação Manual nas Pastas do Sistema

Requer acesso root ou um gerenciador de arquivos com permissões especiais

  1. Instale um gerenciador de arquivos como "ES File Explorer" ou "Files by Google"
  2. Ative a opção "Mostrar arquivos ocultos" nas configurações do gerenciador
  3. Navegue até o caminho abaixo:
/storage/emulated/0/games/com.mojang/
  1. Dentro dessa pasta, você encontrará:
resource_packs - Para pacotes de recursos
behavior_packs - Para pacotes de comportamento
  1. Se você baixou um .mcpack, renomeie para .zip e extraia usando o gerenciador de arquivos
  2. Copie a pasta extraída para resource_packs ou behavior_packs
  3. Feche completamente o Minecraft (remova dos apps recentes)
  4. Abra o Minecraft novamente
  5. Ative o addon nas configurações do mundo
Caminho alternativo: /Android/data/com.mojang.minecraftpe/files/games/com.mojang/ (pode variar conforme a versão)

Método 4: Instalação Manual

  1. Abra o navegador Safari no seu iPhone/iPad
  2. Navegue até o site que contém o arquivo .mcpack
  3. Toque no link de download do arquivo .mcpack
  4. O arquivo será baixado e aparecerá no canto superior direito da tela
  5. Toque no ícone de download para abrir a lista de downloads
  6. Toque no arquivo .mcpack baixado
  7. Uma opção aparecerá: "Compartilhar" → Toque nela
  8. Na lista de opções, procure e toque em "Copiar para Minecraft"
  9. O Minecraft abrirá automaticamente e importará o addon
  10. Vá em "Jogar" → Selecione um mundo → "Editar" → Ative o addon
Este método funciona perfeitamente no iOS sem necessidade de jailbreak

Método 5: Instalação via .mcaddon

  1. Baixe o arquivo .mcaddon no seu dispositivo
  2. Renomeie a extensão de .mcaddon para .zip
  3. Extraia o arquivo ZIP usando um aplicativo de extração (Android) ou Files (iOS)
  4. Dentro da pasta extraída, você encontrará:
behavior_packs - Contém o pacote de comportamento
resource_packs - Contém o pacote de recursos
  1. Para Android: Use o método 3 para copiar as pastas para o diretório do Minecraft
  2. Para iOS: Use o app "Files" para acessar a pasta do Minecraft (se disponível) ou use o método 4 para cada pacote separadamente
  3. Feche e reabra o Minecraft
  4. Ative ambos os pacotes no mundo nas configurações

Método 6: Instalação em Mundo Específico

  1. Abra o Minecraft e vá em "Jogar"
  2. Selecione o mundo onde deseja instalar o addon
  3. Toque no ícone de "Editar" (lápis) ao lado do mundo
  4. Role para baixo e toque em "Abrir Pasta do Mundo" ou "World Folder"
  5. Uma pasta será aberta no gerenciador de arquivos. Dentro dela, você verá:
behavior_packs - Para addons de comportamento do mundo
resource_packs - Para addons de recursos do mundo
  1. Copie a pasta do addon para a pasta correspondente
  2. Volte ao Minecraft e ative o addon nas configurações do mundo
Este método instala o addon apenas naquele mundo específico, economizando espaço
